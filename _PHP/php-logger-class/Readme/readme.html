Classe de log PHP

Type : Code PHP
Date : 22/02/2012
Version : 1
Auteur : www.finalclap.com

Cette archive contient le résultat du tutoriel "Classe de Log en PHP " disponible sur www.finalclap.com à cette adresse :
http://www.finalclap.com/tuto/php-logger-class-78/.

Le répertoire www contient la classe de log (dans lib/Logger.class.php), ainsi qu'un script de test (index.php).
Pour tester, il suffit de placer ce dossier www dans le répertoire "DocumentRoot" de votre serveur Apache/PHP (WAMP, MAMP, EasyPHP...), par exemple : C:\EasyPHP5.2.10\www\test-classe-log\, puis d'accéder au script via un navigateur (ex: http://localhost/test-classe-log/www/index.php).

Configuration requise :
Cette classe a été écrite en PHP 5, elle n'est pas compatible avec les versions antérieures de PHP (qui offraient un support beaucoup plus restreint de la Programmation orientée objet).
